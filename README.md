Deep-Dive Analysis & Interactive Dashboarding
Project Overview

This project performs a deep-dive analysis on sales data and builds a simple interactive dashboard using Python. It calculates important business KPIs and visualizes the data with charts.

Objectives
Analyze sales performance.
Calculate Key Performance Indicators (KPIs).
Visualize sales and profit trends.
Build a simple dashboard using Python.
Technologies Used
Python
Pandas
Matplotlib
Dataset

The project uses a sample sales dataset containing:

Customer Name
Sales
Profit
Orders
Key Performance Indicators (KPIs)

The program calculates:

Total Sales
Total Profit
Total Orders
Average Order Value (AOV)
Visualizations

The dashboard includes:

Bar Chart – Sales by Customer
Line Chart – Profit by Customer
Pie Chart – Sales Distribution
